## 4. 数据下载格式要求：XML

### 为什么选择 XML？

从 PMC 下载论文时，**必须使用 XML 格式**（PMC Full-Text XML），而非 PDF，原因如下：

1. **结构化解析**：XML 保留了论文的完整语义结构（标题、摘要、章节、表格、参考文献），便于程序化提取和清洗。
2. **高效文本提取**：无需 OCR 或 PDF 解析库，直接用 `lxml` / `BeautifulSoup` 解析即可获得干净文本。
3. **元数据完整**：XML 中包含 PMCID、DOI、作者、期刊、发表日期等结构化元数据，可直接用于构建 instruction 数据集的标注信息。
4. **表格与公式保留**：XML 中的 `<table-wrap>` 和 `<disp-formula>` 标签可精准定位表格和公式内容，对食品科学中的实验数据提取尤为关键。
5. **批量处理友好**：PMC OA Bulk Download 提供的就是 XML 格式（`.nxml`），可直接编程批量下载和处理。

### 下载方式

- **PMC OA Bulk Download**：https://ftp.ncbi.nlm.nih.gov/pub/pmc/oa_bulk/ — 批量获取 `.tar.gz` 包，内含 `.nxml` 文件。
- **E-utilities API**：使用 `esearch` + `efetch` 按检索策略查询并下载单篇 XML。
- **PMC OAI-PMH**：适合增量更新，获取最新收录的 OA 文章。

### 典型 XML 清洗流程

```
原始 .nxml → 解析 <body> 提取正文段落 → 去除 <ref-list> 参考文献 → 提取 <abstract> 摘要
→ 提取 <table-wrap> 表格 → 合并为结构化 JSON/JSONL → 用于 instruction tuning 数据构建
```

## 5. Agent 执行清单

1. 逐条执行上述 10 个检索策略，通过 NCBI E-utilities 查询 PMC。
2. 对每条策略记录命中的 PMCID 列表并去重。
3. 以 **XML 格式**批量下载全文（优先使用 PMC OA Bulk FTP，辅以 efetch API）。
4. 下载后按领域（策略编号）分目录存储原始 XML。
5. 运行数据清洗 pipeline，将 XML 解析为结构化 JSONL。
6. 补充从 OAPEN / DOAB / MDPI Books 获取的书籍数据（如适用）。
